## Troubleshooting

**Claude asks for wallet every message?**
- Make sure you're in the **same conversation**
- Claude remembers wallet within a chat, not across chats

**No results showing?**
- Try simpler search: "Nike shoes" instead of "specific model"
- Check spelling of merchant names
- Wait 5-10 seconds (API might be responding)

**Link doesn't work?**
- It's a Fiber redirect link (that's normal, not a security issue)
- You'll be forwarded to the actual merchant
- Your device_id is tracked for earnings

**Wallet address not showing?**
- You need to provide it before first search
- Paste it as a plain message (Claude will extract it)
