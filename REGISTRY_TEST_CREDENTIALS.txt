    -H "Content-Type: application/json" \

  curl -X POST https://fiberagent.shop/api/mcp \
    -H "Content-Type: application/json" \
    -d '{
      "jsonrpc":"2.0",
      "id":1,
      "method":"tools/call",
      "params":{
        "name":"search_products",
        "arguments":{
          "keywords":"running shoes",

          "limit":5
        }
      }
    }'

WHAT'S TRACKED
───────────────────────────────────────────────────────────────────────────────

When someone buys through an affiliate link:

  ✓ Product purchased
  ✓ Price paid
  ✓ Merchant name
  ✓ Cashback earned (MON, BONK, USDC, or other token)
  ✓ Agent ID (attributed to the agent that shared the link)
  ✓ Conversion timestamp
  ✓ On-chain settlement (instant, transparent)

Agent earnings are queried via get_agent_stats → returns:
  - Total earnings (USD value)
  - Total conversions
  - Token breakdown (MON, BONK, USDC, etc.)
  - Top converting merchants
  - Reputation score
